这一代人的学习

发表于 2017-07-02 | 分类于 随笔 | 阅读次数

也许从开始学习的时候，我就从未认真思考过这个问题，我该如何学习？虽然我经常告诉我自己和我的女朋友，学习就是因为不会，所以你才会花费大量的时间去学习，而不是在你想在地方干自己想干的事情。直到今天，直到前不久刚看完罗辑思维的这一代人的学习，才好像明白了些什么。

其实很久以前有在小米的电台里听过罗胖的罗辑思维，只是后来就没怎么关注了。这次是老师在谈到我们该如何去学习，又该以怎样的方式来学习前端时，建议我们看一看或者听一听罗辑思维的这一代人的学习，所以我特意的找出来，仔细的看了一遍。实话说，看完以后触动很大，感慨颇深。也在突然间，发现有些曾经烦恼的问题突然间迎刃而解。我一向自标自己是一个秩序感十强的人，就是说如果哪里出了错，我可能就会耗在那去想法解决问题，而不会暂时搁置，然后走下一步。这种观念在自学前端的时候，我发现简直是灾难，我很喜欢刨根问底，在遇见不明白的问题代码时，会有N多的为什么。虽然有很多人也经常说没有为什么，记住就好，很多东西都是约定俗成的事情，但我还是会在这些问题上耗费大量的时间与精力。直到两个多月前，决心从头再来的时候，我才强迫自己，心里暗示自己忘掉过往，抛弃那些没必要的包袱，谦逊的从头开始。

我这人有个优点，就是我喜欢没人的时候自己瞎想，瞎想的时候我一定是特谦逊的那种，所以我很喜欢接受别人的长处而发现自己的缺点并加以改正，所以我并不是一个迂腐，冥顽不灵的人。以前的我总想着，我买些教材，买些视频，闷头苦读，把东西看完了，我也就可以出师了；毕竟以前在学校就是这样过来的。但在学前端的时候，我发现我想的太过理想化了。这让我不禁想起曾经大学的计算机老师常常说的一句话，如果你们也像我一样写几十万行代码后，你就不会再问为什么了，也不会看到程序无从下手了。是啊，这么浅显的道理你只有真正的碰壁了，才明白，原来很久以前我就知道。前端（程序员）是一个需要不断练习，不断学习，不断补充新知识的行业，我以前总会想我把书看完了也就学会了，但只有真正的了解后才明白，在这个行业里永远也没有学成这一说。所以很多时候我们应该从根本上正视自己的学习态度及方法。

罗辑思维的节目里谈到的第一个主题词是学习焦虑，这个问题大家都有，从在学校的时候想着如果以后考不上学该怎样到毕业以后我该干什么，如果没有适合的工作我该怎样生存下去的一个转变。很多时候我们会焦虑，但却总将这些归咎于社会的就业压力及生存压力，那现在来看看是否也是一种知识焦虑呢？答案是肯定的，因为你的知识层次限制了你的就业方向，你的见识决定了很多岗位并不适合你，你也意识到自己的不足之处，却又因为种种原因无法再去选择改变什么，这不得不说是一种无奈与悲哀。曾经的我对寒门再难出贵子的思想是嗤之以鼻的，但越了解社会，越倾向于认同这种观点。原因很简单，家长的知识与见识会潜移默化的影响孩子的发展，我不是说所有的人都是这样，但至少很多都是这样。而罗辑思维里这点也印证了我的观点，知识可以改变人的一生，准确点说应该是知识和见识可以改变人的一生。

过去我们是怎么学习呢？在学校我们只需要认真的读书、学习、做题就好。但是离开学校以后呢？我们是否还会像以前那样的去学习吗？答案是否定的，你不可能再像上学的时候那样有那么大把的时间去学习。换个思维，在学校你只需要学习固定的教材就好，那么当现在你想学习，你需要学习的时候却发现你的选择可以有千百种，那么你又该如何选择呢？这两个问题带来的反思会让你发现，第一，你没有那么多专一的精力去学习；第二，你也没有明确的方向或者目标去学习。而且当面临日益增加的焦虑时，你单纯的学习一个技能已不足以支撑你在以后的生活中可以游刃有余。因此，跨界学习也就必不可少了，但最终我们还是要面对一个问题，该如何学习？

罗辑思维给了大家一个总结或者建议吧：第一：人格特性。以前都是跟书学，但在现在知识大爆炸的时代，你自己已无法正确的选择你该学习的知识时，跟着牛人学可能会更好一点，牛人就是知识的载体，所以我们学习的时候尽可能的和牛人在一起。这点我在上补习班的时候感受颇深，很久以前自学前端的时候就单纯的以为，只要学完HTML+CSS+JS我就可以写出理想的页面，但真正的跟着老师从头再来的时候才发现前端要学的东西不仅仅只是那些，需要学习的还有很多很多！第二：概念，通过不断的搜集新概念来高效的学习。学习的资料有很多，怎么样从那么多的话语中提炼出来你想要的内容，就不那么简单了。第三：缝合，就是将你知道的了解到的知识表达一次。就像我现在这样，写出来，就算表达一次。做笔记也是表达的一种方式，你也可以通过演讲等其他方式来表达。第四：碎片化，利用碎片化的时间，碎片化知识去掌握。比如闲的时候背一个单词，看一篇文章等等。。。第五：目标，你一定要有一个名确的目标，并为努力与坚持。

啰嗦了好多，就好像自言自语，自论自证。在以后闲时翻来看看，也一定是一件趣事。
