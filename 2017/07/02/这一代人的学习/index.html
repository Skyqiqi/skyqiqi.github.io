这一代人的学习

发表于 2017-07-02 | 分类于 随笔 | 阅读次数

也许从开始学习的时候，我就从未认真思考过这个问题，我该如何学习？虽然我经常告诉我自己和我的女朋友，就是因为不会，所以你才会在这里花费大量的时间去学习，而不是在你想在地方工作。直到今天，直到前不久刚看完罗辑思维的这一代人的学习，才好像明白了些什么。

其实很久以前有在小米的电台里听过罗胖的罗辑思维，只是后来就没怎么关注了。这次是老师建议看一看或者听一听，所以我特意的找出来，看了一看。实话说，看完以后触动很大，感慨颇深。也在突然间，发现有些问题好像也迎刃而解。我一向自标自己是一个秩序感十强的人，就是说如果哪里出了错，我可能就会耗在那去想法解决问题，而不会暂时搁置，然后走下一步。这种观念在自学前端的时候，我发现简直是灾难，我很喜欢刨根问底，在遇见不明白的问题代码时，会有N多的为什么。虽然有很多人也经常说没有为什么，记住就好，很多东西都是约定俗成的事情，但我还是会在这些问题上耗费大量的时间与精力。直到两个多月前，决心从头再来的时候，我才强迫自己，心里暗示自己忘掉过往，抛弃那些没必要的包袱，谦逊的从头开始。

我这人有个好处，就是我喜欢没人的时候自己想问题
